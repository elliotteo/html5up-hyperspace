Preproessing for diffusion weighted imaging sequences

Bash script used for MRI can be found in the link above. Below is a summary of the preprocessing pipeline.

Pre-processing

Images from the Seimens scanner in DICOM format were transferred to a Linux workstation and subsequently converted to NIfTI formate using the mrconvert function (MRtrix3). The b = 0 weighted image and its complimentary reverse phase direction were combined to calculate an image distorition map using FSL Topup (version 5.0.6, FMRIBâ€™s library, 2013) that was subsequently used to correct for susceptibility-induced distortions. Brain masks were manually drawn from all piglets on the average of the first b = 0 and reverse phase encoded image using ITKsnap. Individual diffusion weighted images were then corrected for head movement between DWi volumes and the geometric distoritions due to magnetic suseptibillity, inhomogenatities and eddy currents using FSL eddy. After this step, a mean b = 3000 DWI map was created for better spatial contrast, after which the brain mask delineation was repeated for each piglet to obtain a more accurate brain mask.
